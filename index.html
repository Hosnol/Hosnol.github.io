Portfolio

Project Saya

Berikut ini adalah project yang pernah saya kerjakan selama di perkuliahan atau pun project mandiri untuk mengasah kemampuan dalam mengembangkan sebuah website atau aplikasi.

Website Portfolio

Project ini fokus pada pembuatan website portfolio yang stylish dan responsif dengan memanfaatkan kekuatan Tailwind CSS. Desain simpel dan intuitif, ditambah dengan animasi halus, memberikan tampilan yang menarik. Penggunaan komponen Tailwind mempermudah proses pengembangan, sementara optimisasi kinerja meningkatkan kecepatan pemuatan. Hasilnya adalah portfolio yang efisien, memukau, dan mudah dinavigasi.

Books App Back-End

Project ini adalah bagian back-end dari aplikasi Books yang memiliki fitur CRUD (Create, Read, Update, Delete). Aplikasi ini memungkinkan pengguna untuk mengelola daftar buku. Dengan teknologi yang digunakan Node.js untuk Runtime JavaScript server dan Hapi untuk kerangka kerja back-end untuk Node.js

Klasifikasi Ukuan Bibit Ikan Lele

Aplikasi ini merupakan tugas akhir saya (skripsi) yang digunakan untuk mengatasi masalah pada proses grading bibit ikan lele secara manual dengan menggunakan metode Deep Convolutional Neural Network (DCNN) dengan arsitektur MobileNetV2 dan alat OpenMV Cam.

Online Shop

Sebuah simple website yang menerapkan konsep SPA (Single Page Apllication) dengan React Router dan menerapkan JSON API untuk data produk dan ketika memasukkan produk ke keranjang. Memiliki beberapa menu diantaranya Home, List produk, Keranjang dan About.

Monitoring Suhu Air Tambak Udang Berbasis IoT

Proyek ini digunakan untuk melakukan monitoring air pada tambak udang dengan menggunakan sensor DS18B20 untuk mengetahui suhu pada air tambang tersebut. Dari suhu air tersebut maka dapat ditentukan kualitas air baik untuk tambak udang atau tidak, lalu di monitoring melalui hp/website.

Website Restoran

Website restoran ini dikembangkan menggunakan PHP native untuk memberikan pengalaman pengguna yang sederhana dan efisien. Website ini dirancang untuk memenuhi kebutuhan pengguna dengan sederhana, menawarkan informasi yang jelas tentang menu, harga, serta pesanan.
